# Pricing

Nexus Memory System is priced around the **token value it saves**.

## How We Price

Each Nexus recall uses ~30 tokens vs Hindsight's ~500 tokens (94% reduction).
For a 5-agent team, that's **$36/month in token savings** at Sonnet 4.6 pricing.

Our pricing is set below that savings — you always come out ahead.
